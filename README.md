## Installation

The website requires [Gulp.js](https://gulpjs.com/) to run.

Install the dependencies using *Node.js*.

```sh
$ npm i
$ gulp
```

## Tech Stack

- HTML5, CSS3 and JavaScript for sure.
- Sass, Gulp.js
